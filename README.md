# 💰 FinanceX Dashboard

A modern personal finance dashboard built using **HTML, CSS, and Vanilla JavaScript**.

FinanceX helps users manage income, expenses, budgets, and financial insights through a clean, responsive interface with interactive charts and LocalStorage support.



---

## ✨ Features

- 📊 Dashboard Overview
- 💰 Income & Expense Tracking
- ➕ Add Transactions
- ✏️ Edit Transactions
- 🗑️ Delete Transactions
- 🔍 Search Transactions
- 🎯 Budget Tracker
- 📈 Analytics Dashboard
- 🥧 Pie Chart
- 📊 Bar Chart
- 🌙 Dark Mode
- 💾 LocalStorage Support
- 📄 CSV Export
- 📱 Fully Responsive

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
